riŋ — Classical Star Language Dictionary

Definition

riŋ (inflected for 3rd-person nominative divine singular, 3rd-person passive divine singular)
  1. Demonstrative pronoun used far but not so far things; that.
Note: only used when contrast is needed with nyá.
