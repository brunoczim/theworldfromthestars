jáwnya — Classical Star Language Dictionary

Definition

jáwnya (inflected for 1st-person postpositional animate collective)
  1. Demonstrative pronoun used for near things; this.
  2. Demonstrative pronoun used far but not so far things; that.
Sense 2 is only used if there is no contrast between near and far demonstratives.

Romanization

jáwnya
